# settleDown
install nodejs and npm
----------------------
~~~
curl --silent --location https://rpm.nodesource.com/setup_5.x | sudo bash -
yum -y install nodejs
~~~

install git
-----------
~~~
sudo yum install git-all
~~~

install gulp
------------
~~~
npm install --global gulp-cli
~~~
